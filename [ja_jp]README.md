# SsrsPack

高品質な視覚効果、没入感のあるサウンド、そしてパフォーマンスの最適化を確保しつつ、このModパックはゲームコンテンツを深く改造し、全く新しい没入型体験を提供することを目指しています。

## ✨ 主な特徴

豊富な新コンテンツを導入し、**難易度の高い**世界を共に構築しています：
* **新しい装飾**：多種多様な建築・装飾要素で、あなたの拠点をより個性的に。
* **実用的なツール**：生存、探索、戦闘をサポートする専用ツールを追加。
* **未知の生物**：見たことのない生物との遭遇。彼らは味方かもしれないし、致命的な脅威かもしれません。
* **幻想的な地形**：新たに生成されるバイオームや地形を探索。どの風景にも謎と挑戦が隠されています。

## ⚠️ 開発状況 (Alpha版)

**注意：このModパックは現在Alpha版です。** 主要コンテンツはまだ完成しておらず、プレイ中にバグに遭遇する可能性があります。さらなるコンテンツは現在も開発・改善中です。このModパックの成長にぜひご参加ください！
